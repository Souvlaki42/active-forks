# Active Forks

A Next.js app that shows the active forks of a GitHub repository.

## Getting Started

First, run the development server:

```bash
pnpm dev
# or
yarn dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
